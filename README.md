# Webtech2_XUXEJO_Beadando

Adatbázis configolása:

backend/config/config.json helyen a file-ban átírni az adatbázis adatokat

az én esetemben MySQL-t használtam, MySQL Workbenchet.


{

    "host": "localhost",
  
    "user": "root",
  
    "database": "posts",
  
    "password": "< password >"
  
}

Szerver elindítása:

cd backend

$ npm start

cd ../frontend

$ npm start
